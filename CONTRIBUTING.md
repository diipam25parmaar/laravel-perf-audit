# Contributing to laravel-perf-audit

Thank you for wanting to contribute! Please read the following guidelines before opening a PR.

- Fork the repository and create a branch per feature/bugfix.
- Follow PSR-12 coding style.
- Run `composer install` and `vendor/bin/phpunit` locally.
- For substantial changes, open an issue first describing the problem and the high-level solution.
- All pull requests must include tests where applicable.
